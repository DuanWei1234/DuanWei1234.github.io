Human4DiT

Created2024-07-16|Updated2024-07-20
|Post Views:

Abstract

  1. 将UNet于控制模块几何，实现高质量自由视点和时空一致长人类视频的合成。
  2. 提出4D diffusion transformer架构，其包含3个transformer处理2D图像、时间和视点。
  3. 收集了多维度数据集，并制定训练策略来利用各种模态的数据集
  4. 提出了一种时空一致的扩散采样策略来生成一致的360度长的人类视频。

Network Structure

网络核心为一个4D diffusion transformer和控制条件注入模块。整体4D diffusion transformer采取了一个级联结构，包括2D图像、时间和视图transformer模块。此外，条件注入模块包括参考图像注入模块、SMPL注入模块、时间注入模块和视点注入模块。参考图像注入模块和SMPL注入模块利用UNet架构来确保控制条件的像素对齐注入。

4D Diffusion Transformer

输入为多视角视频latents，其可以变为5D token xt∈RV×T×H×W×Cx_t\in \mathbb{R}^{V\times T\times H\times W\times C}xt​∈RV×T×H×W×C，其中C为每个token的特征维度，V为视点维度。
首先图像输入到2D图像transformer block，其捕捉每一帧内的空间self-attention。

xts=rearrange(xt,(V×T,H×W,C))x_t^s=rearrange(x_t,(V\times T,H\times W,C)) xts​=rearrange(xt​,(V×T,H×W,C))

Qs,Ks,Vs=fsQ(xts),fsK(xts),fsV(xts)Q_s,K_s,V_s=f_s^Q(x_t^s),f_s^K(x_t^s),f_s^V(x_t^s) Qs​,Ks​,Vs​=fsQ​(xts​),fsK​(xts​),fsV​(xts​)

x^ts=softmax(QsKsTdk)Vs\hat{x}_t^s=softmax(\frac{Q_sK_s^T}{\sqrt{d_k}})V_s x^ts​=softmax(dk​​Qs​KsT​​)Vs​

上面公式先将张量变换为3D，第一个维度代表V×TV\times TV×T代表batchsize，在后两个维度上执行attention操作，相当于对图像执行attention。
接着输入到视点transformer block

xtv=rearrange(x^s,(T,V×H×W,C))x_t^v=rearrange(\hat{x}_s,(T,V\times H\times W,C)) xtv​=rearrange(x^s​,(T,V×H×W,C))

Qv,Kv,Vv=fvQ(xtv),fvK(xtv),fvV(xtv)Q_v,K_v,V_v=f_v^Q(x_t^v),f_v^K(x_t^v),f_v^V(x_t^v) Qv​,Kv​,Vv​=fvQ​(xtv​),fvK​(xtv​),fvV​(xtv​)

x^tv=softmax(QvKvTdk)Vv\hat{x}_t^v=softmax(\frac{Q_vK_v^T}{\sqrt{d_k}})V_v x^tv​=softmax(dk​​Qv​KvT​​)Vv​

在这一步直接对所有视点的2D空间维度进行attention操作，这一步由于计算量大会限制视点维度的大小，在推理阶段中解决。最后输入到时间transformer block中捕捉帧间关系。

xtm=rearrange(x^v,(V×H×W,T,C))x_t^m=rearrange(\hat{x}_v,(V\times H\times W,T,C)) xtm​=rearrange(x^v​,(V×H×W,T,C))

Qm,Km,Vm=fmQ(xtm),fmK(xtm),fmV(xtm)Q_m,K_m,V_m=f_m^Q(x_t^m),f_m^K(x_t^m),f_m^V(x_t^m) Qm​,Km​,Vm​=fmQ​(xtm​),fmK​(xtm​),fmV​(xtm​)

x^tm=softmax(QmKmTdk)Vm\hat{x}_t^m=softmax(\frac{Q_mK_m^T}{\sqrt{d_k}})V_m x^tm​=softmax(dk​​Qm​KmT​​)Vm​

这四个transformer组成一个单独的4D transformer block，整个网络架构由级联4D transformer block组成。

Control Modules

  1. Camera Control Module
    假设输入的第一帧图像的相机c1c_1c1​为世界坐标系，提取其中的旋转矩阵作为9D tensor r1r_1r1​并使用位置编码

yc=(sin(20πr1),cos(20πr1),...,sin(2L−1πr1),cos(2L−1πr1))y_c=(sin(2^0\pi r_1),cos(2^0\pi r_1),...,sin(2^{L-1}\pi r_1),cos(2^{L-1}\pi r_1)) yc​=(sin(20πr1​),cos(20πr1​),...,sin(2L−1πr1​),cos(2L−1πr1​))

其它帧的相机参数{ci=2V}\{c^V_{i=2}\}{ci=2V​}计算为第一帧的相对旋转矩阵，并进行位置编码。将这些位置编码使用MLP fcf_cfc​编码到CLIP image相同维度的embedding，并通过attention注入到view transformer module中，其跟在self-attention之后。

xtv′=xtv+fc(yc)x_t^{v'}=x_t^v+f_c(y_c) xtv′​=xtv​+fc​(yc​)

  1. Temporal Embedding Module
    对时间维度(帧数)使用位置编码，如上一样得到ymy_mym​，接着使用MLP fmf_mfm​进行编码，加在temporal transformer self-attention中：

xtm′=xtm+fm(ym)x_t^{m'}=x_t^m+f_m(y_m) xtm′​=xtm​+fm​(ym​)

  1. SMPL Control Module
    具体先获得每帧人中的顶点，并使用相机参数cvc_vcv​渲染出法线图M_n(i,v)。将法线图与相机参数逆矩阵rv−1r^{-1}_vrv−1​解耦出SMPL信息。接着使用UNet EpE_pEp​从渲染图像中提取特征yny_nyn​，这些特征最终与输入特征x_t进行合并，送入到4D Diffusion Transformer中，保证像素对齐的、一致性的、准确的人体运动信息被注入到网络中。
  2. Human Identity Reference Module
    首先从参考图像中提取身份信息，并用UNet EidE_{id}Eid​提取身份信息。与输入xtx_txt​一起输入4D Diffusion transformer；此外，使用CLIP提取图像embedding yey_eye​通过cross-attention将身份信息注入到每个transformer block中的self-attention后。

Q,K,V=fQ(x^t),fK(ye),fV(ye)Q,K,V=f^Q(\hat{x}_t),f^K(y_e),f^V(y_e) Q,K,V=fQ(x^t​),fK(ye​),fV(ye​)

x~t=softmax(QKTdk)V\tilde{x}_t=softmax(\frac{QK^T}{\sqrt{d_k}})V x~t​=softmax(dk​​QKT​)V

Dataset and Training Strategy

对于2D数据集，只训练2D transformer，使用CLIP图像embedding作为人物身份条件。
对于单视图视频数据集，训练2D和时间transformer，随机选择视频中的一帧作为参考图像。
对于多视角视频，一同训练所有transformers，随机从多视角数据中选择一张图片作为参考图像。
对于3D数据集，只训练2D和view transformers，随机选取一张渲染视角作为参考。
对于4D数据集，将其渲染为密集视角视频或者连续的视角移动数据，训练所有transformers。

Efficient Spatial-Temporal Sampling

由于输入的数据窗口大小为时间T×视点V，作者提出了一个时空一致性采样方法。
要生成自由视角视频作为一个单目视频序列，最大化时间窗口保证长期时间一致性；第二个策略为将自由视角视频当作多视角短视频几何，使用更大的视点窗口和更小的时间窗口，维持多视角的一致性。

从全局来看，它利用视图转换器来保持大视点分离之间的一致性。对于相邻视点之间较小的帧间运动，时间变压器确保了相干性。最终，该方法在有限的输入窗口约束下，实现了长时间自由视点的人体运动视频的时空一致生成。
