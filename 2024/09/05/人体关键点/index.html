json文件输出

输出形式通常以json文件形式，包括pose_keypoints_2d、face_keypoints_2d, hand_left_keypoints_2d 和 hand_right_keypoints_2d， 格式为x1,y1,c1,x2,y2,c2,…,坐标 x 和 y 被归一化到 [0, 1], [-1, 1], [0, source size], [0, output size] 等范围，其取决于 keypoint_scale flag. c 是范围在 [0, 1] 的置信度分数(confidence score).
开启–3d后，输出为x1,y1,z1,c1,x2,y2,z2,c2,… 其中，c 值为 0 或 1，取决于 3D reconstruction 是否成功.

heatmap输出

输出格式为widthnum_heatmapsheight的一个大矩阵，其中heatmaps=67，18 body parts+1 bg+19*2PAFs
